# Username Tier System

Status: Draft
Last Reviewed: 2026-02-27

## Scope

Defines fallback rules when "tinman" is unavailable.

---

## Tier Escalation

Tier 0: tinman  
Tier 1: tinmanlab  
Tier 2: tinmanforge  
Tier 3: tinmansystems  
Tier 4: tinmanworks  
Tier 5: tinmanstudio  

---

## Rules

1. Always attempt Tier 0 first.
2. If unavailable, move sequentially.
3. Never skip tiers.
4. Never introduce unlisted suffixes.
5. Avoid numbers unless absolutely unavoidable.

---

## Principle

Consistency across platforms is more important than identical usernames everywhere.
